Privacy Policy Leave Me Alone

Leave Me Alone reads the number of incoming calls to determine if the number is to be blocked. It does not store incoming numbers nor does send it them to a remote server.
Leave Me Alone stores the phone numbers entered by the user on the phone. All numbers are only stored for the purpose of comparing them against incoming number to decide if a incoming call should be blocked. No phone numbers are transmitted to a remote server.
Leave Me Alone searches the phones contacts if the option Only allow contacts is enabled to decide if the incoming call is from a known number and should be allowed to ring. No contact information is stored within the app or transmitted to a remote server.
Leave Me Alone has access to the internet to download the latest block lists. No personal data is transmitted to a remote server.
